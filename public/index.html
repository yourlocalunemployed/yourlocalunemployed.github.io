  • Runs Claude Code & Claude Desktop

pfSense VM

  • NAT adapter
  • Host-only adapter

Explanation of Setup

Both Claude Code and Claude Desktop run on the Debian VM and have designated access to the file system within it. The pfSense VM acts as a firewall and virtual router specifically for the Debian machine — the LAN card on pfSense acts as the WAN interface and the host-only card acts as the LAN interface.
